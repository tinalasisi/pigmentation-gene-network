Pigmentation gene-network rescue screen

Project website — a guided walkthrough of the finding and how it was built. This page is a placeholder; the real site is assembled here by Claude Code.

What this is

A convergence-graded rescue screen: pigmentation loci their own authors reported but could not mechanistically explain, resolved to a causal gene and tested for whether they connect into the melanogenesis network — each graded by converging evidence, with a named confirming experiment.

Planned narrative (site beats)

  1. The problem — GWAS strands pigmentation loci it can't explain.
  2. A trustworthy substrate — the identity-resolved melanogenesis network (and why the network you choose changes the answer).
  3. Validation & gaps — a CRISPR screen both confirms the networks and exposes 142 missing genes.
  4. The rescue — the author-unexplained loci connected to melanogenesis; the hero locus.
  5. The graded catalog + confirming experiments + honest limits.
