A seven-step pipeline that downloads 117 primate genomes, extracts and aligns 110 pigmentation and hormone genes, and tests each for natural selection on a supercomputer.

Comparative genomics · how it works

Testing 110 primate genes for natural selection

Some primates are dichromatic — the sexes or ages differ in color. Have the genes behind color been shaped by selection differently in these species? And across the ~14 independent origins of dichromatism, did evolution reuse the same genes or different ones? We compare pigmentation genes against hormone genes across 117 species.

The pipeline · these 7 steps run for all 110 genes (117 species each)

  1. 1

    🧬Download the genomes NCBI

    Fetch the full DNA of 117 primate species (~700 GB). Done once, reused everywhere.

    117 giant instruction books — each ~2.6–3 billion letters (sizes vary by species).

  2. 2

    ✂️Find & cut out each gene miniprot

    For every one of the 110 genes, locate it inside all 117 genomes and extract its DNA. Where there's no gene map, match the human protein to find it.

    Finding one short passage inside a 3-billion-letter book — 110 passages, 117 books.

  3. 3

    📊Line the copies up MAFFT

    For each gene, stack all 117 species' versions so matching positions line up, adding gaps where evolution inserted or deleted pieces.

    117 translations of one paragraph, aligned word-by-word — done for every gene.

  4. 4

    🌳Bring in the family tree phylogeny

    Use the known evolutionary tree so the math knows which species are close cousins and which are distant.

    Cousins resemble each other for free — the tree corrects for that.

  5. 5

    🔬Test for selection RELAXaBSREL

    For each gene, compare protein-changing DNA changes vs. silent ones. RELAX asks if selection is intensified or relaxed in dichromatic species (a number K); aBSREL asks which branches had bursts of change.

    Is something actively pushing this gene to change — and where?

  6. 6

    🔀Compare the independent origins per-origin

    Run the test separately for each origin of dichromatism and ask: the same genes every time, or different ones?

    Did evolution reach for the same toolbox, or improvise each time?

  7. 7

    📋Collect & label results

    Gather every gene's K-values, p-values and branch results into tables, each gene tagged by its functional group.

    The raw output for all 110 genes, made readable for the biology.

Two things that make it work

🖥️

It runs on a supercomputer, in parallel

On the Great Lakes cluster, all 110 genes run at the same time as a job array — so the total time equals the single slowest gene, not the sum. That's why the two giant genes get their own track: one slow gene shouldn't hold up the other 108.

✓

"Completed" is not "correct"

A supercomputer reports a job as done even when it produced nothing. Every stage's actual output is checked before it's trusted — and failures are made loud (a red FAILED task), not silently hidden.
